    Congratulations on making it to the RAW verification process! For each photo, please upload one or more RAW files (.arw, .cr2, .nef, etc.) for judging verification. If you have more than five files for your image (e.g. if it is a panorama/exposure blend/focus stack/etc), please select the five that best represent your photograph. We may request the rest later. To submit, first click on “Choose file” and add each file (up to 5 per final photo), and then click “Update.” Then files will then upload, which may take a few minutes depending on your connection. If you have any trouble please contact us at info@naturallandscapeawards.com

    When you press "update", any files that have the 'clear' box ticked will be removed. You don't need to use this facility if you are just replacing files.

    If you can see the name of your raw file(s) next to your entry after youu have pressed 'update', then the raw files have been successflly uploaded. You can check for sure by reopening the page to see if the raw file is still listed next to your image.

    If you are having problems uploading all of your files in one go, please try uploading in smaller batches or one file at a time. Our upload system uses Amazon S3 file storage which can be slow. Many thanks for your patience!

    Please email us if you have any questions.
